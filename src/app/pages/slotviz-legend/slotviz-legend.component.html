Block States

Each block has two states. The border color represents the network status while the content color indicates your validator's status. This makes it easy to backtrack if a missed duty was related to a network issue or an issue on your end.

Content Color Scheme

Green = All validator duties successful
Red = All validator duties failed
Yellow = Some duties successful / some failed
Empty = Some duties are still pending

Border Color Scheme

Green = Network proposed block
Red = Network missed block

Symbols

Block Proposal Duty

Attestation Duty

Sync Committee Duty

Slashing

Multiple duties / events in this block

You can click on that block for a breakdown

Examples

Green content with green border:
Network proposed a block and you successfully attested to it

Red content with red border:
You missed your block proposal

Yellow content with green border:
Network proposed the block, but only some of your validators successfully attested to it.

Green content with red border:
Network missed the block but your duties were successful

Green content with green border:
Network proposed and you caught someone else's validator cheating and slashed them

Red content with green border:
Network proposed but your validator has been slashed.
